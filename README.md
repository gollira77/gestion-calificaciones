# Gestión de Calificaciones

Programa por consola desarrollado en **JavaScript con Node.js** para gestionar calificaciones de alumnos.

---

## Funcionalidades

- Ver alumnos y sus materias  
- Agregar alumnos  
- Agregar o modificar notas  
- Calcular promedio por alumno  
- Mostrar el alumno con mejor promedio  
- Ordenar alumnos por promedio  

---

## Tecnologías

- JavaScript  
- Node.js  
- Consola  

---

## Requisitos

Tener instalado:

- Node.js  
- Python 3 (solo para la versión Python)

---

## Ejecución (Versión JavaScript)

```bash
npm install
npm start
```

## Versión Python

```bash
VersionPython/
```

No tiene datos precargados
Comienza con lista vacía
Guarda los datos en archivo JSON
Mantiene persistencia entre ejecuciones

## Estructura versión Python

```bash
VersionPython/
│
├── main.py
│
└── data/
    └── alumnos.json
```

El archivo alumnos.json comienza vacío:

```bash
[]
```

Los alumnos cargados por consola se guardan automáticamente.

## Cómo ejecutar la versión Python

Primero entrar a la carpeta:

```bash
cd VersionPython
```

Luego ejecutar:

```bash
python main.py
```
